# ButterCMS Sinatra Example

## Documentation

For a comprehensive list of examples, check out the [API documentation](https://buttercms.com/docs/api/).

## Get Started

Start server with [foreman](https://github.com/ddollar/foreman):

```
$ foreman start
```
